Local assets integration
=======================

All image dependencies were rewritten to local files under `assets/images/`.

Folders:
- `assets/images/items/`
- `assets/images/path_trees/`
- `assets/images/heroes/`
- `assets/images/runewords/`
- `assets/images/ui/`
- `assets/images/misc/`

Source mapping came from `download_report.csv` and the downloaded `imagenes.zip` package.

If you need to replace or add images later, keep the same relative structure and filenames.
